1. VISÃO GERAL DO PROJETO
O que é o TRAMPO?
Plataforma brasileira de matchmaking de carreiras com Inteligência Artificial. Conecta candidatos a vagas compatíveis automaticamente, sem precisar enviar currículo um por um.
Problema que resolve:

Candidatos perdem horas enviando currículos manualmente
Recrutadores recebem centenas de currículos desalinhados
Vagas somem da internet sem o candidato saber

Solução:

IA lê o currículo e preenche o perfil automaticamente
Sistema calcula % de compatibilidade candidato-vaga
Envia currículo automaticamente para vagas compatíveis
Recrutador recebe ranking dos melhores candidatos

```

```

```

GET  /api/jobs/{id}          → Ver vaga específica
POST /api/jobs/              → Publicar vaga (recrutador)
PUT  /api/jobs/{id}          → Editar vaga
DELETE /api/jobs/{id}        → Deletar vaga

```
POST /api/applications/      → Se candidatar a uma vaga
GET  /api/applications/      → Listar candidaturas da vaga (recrutador)

1. Acessa `render.com` → New Web Service
2. Conecta o GitHub `ContaTrampo/trampo`
3. Root Directory: `backend`

6. Adiciona todas as variáveis de ambiente da seção 3

### Frontend (Vercel):
1. Acessa `vercel.com` → Import Project
2. Conecta o GitHub `ContaTrampo/trampo`
3. Root Directory: `frontend`
